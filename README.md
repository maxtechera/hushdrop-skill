# hushdrop (Agent Skill)

The Hushdrop agent skill — install it so your AI agent can publish what it builds as a private,
branded link on your own domain.

```bash
npx skills add maxtechera/hushdrop-skill
```

Works with Cursor, Codex, Gemini CLI, Claude Code, and 20+ Agent Skills hosts. The skill itself
is `SKILL.md`. Full project: **https://hushdrop.dev** · https://github.com/maxtechera/hushdrop · MIT.
